Usage: gdrv <command> [flags]

gdrv is a command-line tool for interacting with Google Drive. It supports file
operations, folder management, permissions, and more.

All commands support JSON output for automation and scripting.

Flags:
  -h, --help                      Show context-sensitive help.
      --profile="default"         Authentication profile to use
      --drive-id=STRING           Shared Drive ID to operate in
      --output="json"             Output format (json, table)
  -q, --quiet                     Suppress non-essential output
  -v, --verbose                   Enable verbose logging
      --debug                     Enable debug output
      --strict                    Convert warnings to errors
      --no-cache                  Bypass path resolution cache
      --cache-ttl=300             Path cache TTL in seconds
      --include-shared-with-me    Include shared-with-me items
      --config=STRING             Path to configuration file
      --log-file=STRING           Path to log file
      --dry-run                   Show what would be done without making changes
  -f, --force                     Force operation without confirmation
  -y, --yes                       Answer yes to all prompts
      --json                      Output in JSON format (alias for --output
                                  json)

Commands:
  version [flags]
    Print the version number

  about [flags]
    Display Drive account information and API capabilities

  files [flags]
    File operations

  folders [flags]
    Folder operations

  auth [flags]
    Authentication commands

  permissions (perm) [flags]
    Permission operations

  drives [flags]
    Manage Shared Drives

  sheets [flags]
    Google Sheets operations

  docs [flags]
    Google Docs operations

  slides [flags]
    Google Slides operations

  admin [flags]
    Google Workspace Admin SDK operations

  changes [flags]
    Drive Changes API operations

  labels [flags]
    Drive Labels API operations

  activity [flags]
    Drive Activity API operations

  chat [flags]
    Google Chat operations

  sync [flags]
    Sync local folders with Drive

  config [flags]
    Configuration management

Run "gdrv <command> --help" for more information on a command.
