Usage: gdrv files <command> [flags]

File operations

Flags:
  -h, --help                      Show context-sensitive help.
      --profile="default"         Authentication profile to use
      --drive-id=STRING           Shared Drive ID to operate in
      --output="json"             Output format (json, table)
  -q, --quiet                     Suppress non-essential output
  -v, --verbose                   Enable verbose logging
      --debug                     Enable debug output
      --strict                    Convert warnings to errors
      --no-cache                  Bypass path resolution cache
      --cache-ttl=300             Path cache TTL in seconds
      --include-shared-with-me    Include shared-with-me items
      --config=STRING             Path to configuration file
      --log-file=STRING           Path to log file
      --dry-run                   Show what would be done without making changes
  -f, --force                     Force operation without confirmation
  -y, --yes                       Answer yes to all prompts
      --json                      Output in JSON format (alias for --output
                                  json)

Commands:
  files list [flags]
    List files

  files get <file-id> [flags]
    Get file metadata

  files upload <local-path> [flags]
    Upload a file

  files download <file-id> [flags]
    Download a file

  files delete <file-id> [flags]
    Delete a file

  files copy <file-id> [flags]
    Copy a file

  files move --parent=STRING <file-id>
    Move a file

  files trash <file-id>
    Move file to trash

  files restore <file-id>
    Restore file from trash

  files revisions list <file-id>
    List file revisions

  files revisions download --revision-output=STRING <file-id> <revision-id>
    Download a specific revision

  files revisions restore <file-id> <revision-id>
    Restore file to a specific revision

  files list-trashed [flags]
    List trashed files

  files export-formats <file-id>
    Show available export formats for a file
