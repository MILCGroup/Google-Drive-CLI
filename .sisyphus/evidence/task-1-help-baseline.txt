=== MAIN HELP ===
gdrv is a command-line tool for interacting with Google Drive.
It supports file operations, folder management, permissions, and more.

All commands support JSON output for automation and scripting.

Usage:
  gdrv [command]

Available Commands:
  about       Display Drive account information and API capabilities
  activity    Drive Activity API operations
  admin       Google Workspace Admin SDK operations
  auth        Authentication commands
  changes     Drive Changes API operations
  chat        Google Chat operations
  completion  Generate the autocompletion script for the specified shell
  config      Configuration management
  docs        Google Docs operations
  drives      Manage Shared Drives
  files       File operations
  folders     Folder operations
  help        Help about any command
  labels      Drive Labels API operations
  permissions Permission operations
  sheets      Google Sheets operations
  slides      Google Slides operations
  sync        Sync local folders with Drive
  version     Print the version number

Flags:
      --cache-ttl int            Path cache TTL in seconds (default 300)
      --config string            Path to configuration file
      --debug                    Enable debug output
      --drive-id string          Shared Drive ID to operate in
      --dry-run                  Show what would be done without making changes
  -f, --force                    Force operation without confirmation
  -h, --help                     help for gdrv
      --include-shared-with-me   Include shared-with-me items
      --json                     Output in JSON format (alias for --output json)
      --log-file string          Path to log file
      --no-cache                 Bypass path resolution cache
      --output string            Output format (json, table) (default "json")
      --profile string           Authentication profile to use (default "default")
  -q, --quiet                    Suppress non-essential output
      --strict                   Convert warnings to errors
  -v, --verbose                  Enable verbose logging
      --version                  version for gdrv
  -y, --yes                      Answer yes to all prompts

Use "gdrv [command] --help" for more information about a command.
